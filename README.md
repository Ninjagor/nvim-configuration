## Features Added

This configuration has ALL of the latest features of NvChad, + some more.
LSP's installed:
- Pyright
- Tsserver
- PrismaLS
- luals
- jdtls
- omnisharp
- tailwindcss

And has TreeSitter parsers installed for the above languages, + bash, Dockerfile, and yaml.

Bug Fixes:
- Has fix to tsserver's semanticToken bug. Bug fixes inside of `init.lua`, `chadrc.lua` and `lspconfig.lua`

Just clone repo to use.
